2. Проект

Разработка мобильного приложения управления складом дорогих металлов.

3. Задача проекта

Реализация мобильного приложения автоматизированного управления складом.

4. Риски

  • Недостаточная квалификация разработчиков;
  • Различия в ожиданиях клиента.
  • Дефицит ресурсов.

5. Стратегии управления рисками.

  • Стратегии управления рисками.
